== Installation ==
1. Be sure you're running WooCommerce 2.1+ in your shop.
2. First, you need to download the plugin from the source (which will be a zip file). Next, you need to go to WordPress admin area and navigate to Plugins->Add New.
3. After that, click on the Upload Plugin button on top of the page.
4. This will bring you to the plugin upload page. Here you need to click on the choose file button and select the plugin file you downloaded earlier to your computer.
5. After you have selected the file, you need to click on the install now button.
6. WordPress will now upload the plugin file from your computer and install it for you. You will see a success message after the installation is finished.
7. Once installed, you need to click on the Activate Plugin link to start using the plugin.
8. You will need to configure the settings for the plugins with the information provided by Scalapay. (You can find plugin with name 'Scalapay Easy Installment Gateway' in plugins list and click configure)

## Manually Install a WordPress Plugin using FTP

1. First you will need to download the plugin's source file (it will be a zip file). Next, you need to extract the zip file on your computer.
2. Extracting the plugin zip file will create a new folder with the same name. This is the folder that you need to manually upload to your website using an FTP client.
3. Open the FTP client on your computer and connect it to your website using the login credentials provided by your web host. Once connected, you need to access the path /wp-content/plugins/
4. Next, upload the folder you extracted from the zip file to the /wp-content/plugins/ folder on your web server.
5. After uploading the files, you need to visit the WordPress admin area and click on the Plugins link in the admin menu. You will see your plugin successfully installed on the plugins page.
6. You need to click on the Activate link below the plugin. Upon activating, you may need to configure the plugin settings with the information provided by Scalapay. (You can find plugin with name 'Scalapay Easy Installment Gateway' in plugins list and click configure)


== Frequently Asked Questions ==

= Gateway not showing on front-end after installation =
Please go to admin->plugin list->scalapay plugin->configure and see on top of page enable checkbox is checked or not.

= Gateway not showing on product detail page =
Please confirm the product price you are viewing is not more than 300 Euro or euqivalent.
